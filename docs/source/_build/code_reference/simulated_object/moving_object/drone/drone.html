create_xml_element(pos: str, quat: str, color: str) → dict[str, list[Element]][source]¶

It is each object’s responsibility to know how it ought to look in a MJCF file. This method is how those XML elements are generated.

Parameters:

  • pos (str) – The position of the object in the scene, x-y-z separated by spaces. E.g.: “0 1 -1”

  • quat (str) – The quaternion orientation of the object in the scene, w-x-y-z separated by spaces.

  • color (str) – The base color of the object in th scene, r-g-b-opacity separated by spaces, scaled 0.0 to 1.0

Returns:

A dictionary where the keys are tags of XML elements in the MJCF file, and the values are lists of XML elements to be appended as children to those XML elements.

Return type:

dict[str, list[ET.Element]]
